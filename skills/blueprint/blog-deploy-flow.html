Blog Deploy Flow

GitHub Issue → Actions → UPYUN → CDN 自动部署流水线

① 内容创作 & 提交

  • • 使用 Markdown / Org-mode 撰写博客
  • • 通过 GitHub Issue API 提交为新 Issue
  • • Issue 标题作为文章标题，正文即内容
  • • Label 控制分类、发布状态等元信息

② GitHub Actions 自动化

  • • Issue Webhook 监听新建 / 更新事件
  • • Workflow dispatch 触发构建流水线
  • • 静态站点生成 (Hugo/Hexo/Jekyll)
  • • 执行 upx 上传脚本推送至又拍云

③ 又拍云存储

  • • 又拍云对象存储托管静态资源
  • • upx CLI 工具增量同步文件
  • • 自动设置 Content-Type 与缓存头
  • • 支持目录级文件管理与版本回滚

④ CDN 加速 & 访问

  • • 自定义域名 CNAME 解析至又拍云
  • • HTTPS 证书自动管理与续期
  • • 全国 CDN 节点加速内容分发
  • • 终端用户通过域名直接访问博客
