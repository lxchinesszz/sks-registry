微服务部署流水线

Code Commit → CI Build → Registry → Deploy → Monitor 全链路

① 代码提交

  • • Git push 触发 Webhook
  • • PR/MR 合并到主分支
  • • 触发下游 CI Pipeline
  • • 事件驱动，自动执行

② CI 构建

  • • 代码检查、单元测试、安全扫描
  • • 构建 Docker 镜像并打标签
  • • 多阶段构建减小镜像体积
  • • 推送至容器镜像仓库

③ 镜像存储

  • • 存储版本化容器镜像
  • • 支持多架构 (amd64/arm64)
  • • 镜像扫描与签名验证
  • • CD 工具监听新镜像推送

④ 部署发布

  • • GitOps 自动同步集群状态
  • • 滚动更新，零停机部署
  • • Ingress 路由外部流量到服务
  • • TLS 终止与服务发现

⑤ 监控告警

  • • 健康检查探针 (Liveness/Readiness)
  • • 指标采集与日志聚合
  • • Prometheus + Grafana 可视化
  • • 异常告警通知开发者
